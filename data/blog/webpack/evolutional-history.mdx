# 下载趋势

Webpack 早在 2012 年 3 月就启动了项目，在 2016 - 2017 年开始缓慢地获得用户，之后才得到广泛的应用。

![npm-download](/static/images/blog/webpack/npm-downloads.png)

# 版本发布历程

在 Github 的 [Webpack Tags/Release](https://github.com/webpack/webpack/tags?after=v3.0.0) 中可以看到，前期的版本是比较混乱的，在 v3 之后有所好转，之后的 Webpack 更新也基本上了快车道，一年一个版本。

![versions](/static/images/blog/webpack/webpack-version.png)
